# 《读地球的人》纪录片素材研究

## 🎬 项目目的
为纪录片《读地球的人》脚本寻找和分类素材，分为两类：
1. ✅ 可自由使用（无版权纠纷）
2. ⚠️ 有版权风险但值得参考

## 📁 目录结构
```
earth-materials-research/
├── index.html          # 主页面
├── README.md          # 项目说明
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── script.js      # 交互脚本
├── images/            # 素材截图/示例
└── data/
    ├── free-materials.json    # 免费素材库
    └── reference-materials.json # 参考素材库
```

## 🔍 素材来源分类

### ✅ 免费可商用来源
1. **NASA/ESA** - 太空/地球图像
2. **USGS** - 地质/地震数据
3. **NOAA** - 海洋/气象素材
4. **Wikimedia Commons** - 历史/科学图像
5. **Pexels/Unsplash** - 通用素材
6. **Archive.org** - 历史文献
7. **Creative Commons** - 各类CC许可内容

### ⚠️ 有版权但值得参考
1. **Getty Images** - 专业摄影
2. **Shutterstock** - 商业素材
3. **BBC/NHK** - 纪录片片段
4. **Nature/Science** - 科学期刊
5. **专业纪录片** - 参考风格/构图
6. **博物馆馆藏** - 需要授权
7. **学术机构** - 研究资料

## 🛠️ 使用方法
1. 访问 `index.html` 查看素材分类
2. 点击素材查看详细信息
3. 筛选免费/参考素材
4. 获取素材链接和授权信息
