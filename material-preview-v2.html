批量下载工具

使用命令行工具批量下载素材：

# 下载NASA地球图像
wget -r -l1 -H -t1 -nd -N -np -A.jpg https://eoimages.gsfc.nasa.gov/
# 下载Wikimedia Commons地质学图像
# 需要安装wikimedia-downloader
pip install wikimedia-downloader
wikimedia-download --query "geology" --limit 50 --output ./downloads/
# 下载USGS地质样本
curl -o usgs_samples.zip https://www.usgs.gov/products/multimedia-gallery

建议创建下载目录：mkdir -p downloads/{nasa,usgs,wikimedia}
